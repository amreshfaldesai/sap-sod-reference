SAP Authorization & SoD Reference

Segregation of Duties, decoded field by field.

A working glossary of SAP authorization objects, transaction codes, and the function pairs that commonly create Segregation of Duties conflicts — across FI/CO, MM, SD, and HR. Covers SAP ECC and S/4HANA.

FI/CO

Finance & Controlling

Covers Accounts Payable, Accounts Receivable, General Ledger, Asset Accounting, Bank Accounting, and Controlling.

Authorization Objects
F_LFA1_APPVendor master: application area authorization — controls which vendor account groups/fields a user may create or change.Vendor master maintenance
F_LFA1_BUKVendor master: company code authorization — controls which company codes a user may maintain vendor data for.Vendor master maintenance
F_KNA1_APPCustomer master: application area authorization.Customer master maintenance
F_KNA1_BUKCustomer master: company code authorization.Customer master maintenance
F_BKPF_BUKFI document: company code authorization — controls posting/display of accounting documents per company code.Journal/document posting across FI
F_BKPF_BESFI document: posting key authorization — controls which posting keys (debit/credit types) a user may use.Journal posting
F_REGU_BUKPayment program: company code authorization — controls execution of automatic payment runs (F110).Payment processing
M_RECH_BUKLogistics invoice verification: company code authorization for invoice release/blocking.AP invoice release (MRBR)
F_SKA1_BUKGL account master: company code authorization for creating/changing GL accounts.GL master maintenance
A_S_TCODEAsset accounting: transaction-code-level authorization for asset postings.Asset acquisition/retirement
S_TABU_NAMTable maintenance authorization — controls access to configuration tables (tax codes, tolerance groups, exchange rates, etc.).Configuration/master data changes
K_CSKSCost center: authorization for cost center master data maintenance.Cost center maintenance
K_ORDERInternal order: authorization for order master data and settlement.Internal order creation/settlement
K_PCAProfit center accounting: authorization for profit center master and postings.Profit center maintenance

MM

Materials Management / Procurement

Covers the full Procure-to-Pay cycle: Vendor & Master Data, Purchasing, Goods Movement, Invoice & Payment, and Inventory.

Authorization Objects
M_BEST_BSAPurchasing document: authorization by document type — controls creation/change of POs by document category.PO creation/change
M_BANF_BSAPurchase requisition: authorization by document type.Requisition creation
M_BANF_FRGPurchase requisition: release/approval authorization.Requisition release
M_EINK_FRGPurchase order: release/approval authorization — controls who can approve POs under the release strategy.PO release/approval
M_MSEG_WMBMaterial document: goods movement authorization by plant/movement type.Goods receipt, transfers, scrap
M_RECH_WRKInvoice verification: plant-level authorization for logistics invoice posting.Invoice verification (MIRO)
M_MATE_WRKMaterial master: plant-level authorization for material master maintenance.Material master maintenance
M_EINF_EKOPurchasing info record: purchasing organization-level authorization.Info record / source list maintenance
M_ANFR_EKORFQ (request for quotation): purchasing organization-level authorization.RFQ creation and evaluation
M_VERT_BSAOutline agreement (contract/scheduling agreement): authorization by document type.Contract/agreement maintenance
M_ISEG_WMBPhysical inventory document: authorization by plant for count and adjustment postings.Physical inventory count/adjustment
M_LFM1_BUKVendor evaluation: company code-level authorization.Vendor evaluation scoring
QM_ITBQuality management: inspection lot authorization.Quality inspection and batch release

HR

HR / HCM

Covers Employee Master Data, Payroll, Time & Attendance, Compensation & Benefits, Organizational Management, and Recruitment.

Authorization Objects

P_PERNRHR master data: personnel-number-level authorization, often used to restrict/allow access to an employee's own record.Self-service, payroll access control
PLOGPersonnel planning (Organizational Management): authorization for org structure, positions, and jobs.Org management, position assignment
CATSAPPROVECross-Application Time Sheet: authorization for approving time entries.Time approval
S_USER_GRPUser master maintenance: authorization by user group — relevant to who can provision SU01/PFCG access.Access provisioning
